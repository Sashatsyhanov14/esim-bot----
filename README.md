# eSIM Universal Bot Template

Этот репозиторий содержит универсальный шаблон бота для продажи eSIM.
Шаблон легко клонируется для запуска неограниченного числа ботов.

## 🚀 Как запустить нового клона (Бота)

Для каждого нового бота вам потребуется:
1. Новый токен бота от [@BotFather](https://t.me/BotFather).
2. Хостинг (или папка на сервере) для запуска кода бота.
3. Новый проект Supabase (рекомендуется для разделения баз данных), либо просто используйте те же таблицы, но с добавлением колонки `bot_id`, если хотите всё держать в одной базе (в текущей версии предполагается 1 база = 1 бизнес/бот).

### Шаги по клонированию:

1. **Скопируйте папку `bot`**:
   Сделайте копию папки `bot` и назовите её, например, `bot_europe` или `bot_asia`.
   
2. **Настройте переменные окружения**:
   В новой папке создайте файл `.env` на основе `.env.example`:
   ```env
   BOT_TOKEN=новый_токен_от_botfather
   OPENAI_API_KEY=ваш_ключ_openai
   SUPABASE_URL=url_нужной_базы_данных
   SUPABASE_KEY=ключ_нужной_базы_данных
   MANAGER_ID=telegram_id_менеджера_этого_бота
   ```

3. **База данных**:
   В проекте Supabase выполните SQL-скрипт `database_schema.sql`.
   В таблицу `tariffs` добавьте готовые тарифы.
   **Важно**: Вы упомянули "qr коды есть готовые". Просто вставьте прямые ссылки на эти картинки в колонку `payment_qr_url` в таблице `tariffs`. Бот сам их подхватит и отправит клиенту.

4. **Запуск бота**:
   ```bash
   cd bot_europe
   npm install
   node index.js
   ```

### 📱 Клонирование Mini App (Реферальная система)

1. Скопируйте папку `webapp`.
2. Создайте новый проект на Vercel/Netlify.
3. Укажите в Environment Variables (переменных окружения) на хостинге:
   - `VITE_SUPABASE_URL` = URL базы данных конкретного бота
   - `VITE_SUPABASE_ANON_KEY` = ключ базы данных конкретного бота
4. Привяжите полученный URL к кнопке меню нового бота через @BotFather.

## Роли в Mini App (Основатель / Пользователь)
Чтобы сделать человека "Основателем" и дать ему доступ к глобальной статистике в Mini App:
1. Зайдите в базу данных Supabase -> таблица `users`.
2. Найдите нужный `telegram_id`.
3. Измените поле `role` с `user` на `founder`.
